## Usage

1. **Sign In**:
    - Use your GitHub account to log in.
2. **Upload Videos**:
    - Go to NotTube Studio.
    - Select a video file and upload it.
3. **Watch & Interact**:
    - Click on any video to watch it.
    - Like and comment.
4. **Manage Your Content**:
    - Edit video details.
    - View total views in NotTube Studio.
